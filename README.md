# LocalDiscgolf

LocaDiscgolf är ett system för att kunna ha egen lokal statistik på sina discgolfresultat. Syftet är att kunna ha databas, backend och en webbfrontend på en raspberry pi eller liknande och sedan föra resultat på en androidtelefon, som kopplar upp sig mot pi:en.

Syftet är inte att skapa en ersättare för udisc eller tjing, utan fyller istället ett personligt behov av att kunna få statistik på alla familjemedlemmar, även de som inte betala för udisc. Dessutom vill jag ha statistik på webben, så jag kan titta på den i datorn istället för på telefonen. 

I framtiden är tanken att statistiken ska kunna bli mer avancerad vad gäller kastregistrering, tex uppdelning av C1-puttar på andel korta (3-7 meter) och långa (7-10 meter) som sätts. Den ska även kunna skilja på OB och missad mando. Hål ska kunna ha egenskaper som mando, OB, hazard med mera och sedan kan man få procent på hur ofta man missar mando på olika mandohål etc.
